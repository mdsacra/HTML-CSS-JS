#Aula 08 - Operadores (parte 2)

    - Relacionais
    > < >= <= == != 
    "===" este é um operador de Identidade, testa se os valores são idênticos (mesmo valor e mesmo tipo)
    
    - Lógicos
    ! => Negação "Não"
    && => Conjunção "E"
    || => Disjunção "Ou"
    
    - Ternário
    ? :
    Ex.: " média >= 7.0 ? 'Aprovado' : 'Reprovado' "

    
    Primeiro faço os operadores Aritméticos, depois os Relacionais e depois os Lógicos. Entre os operadores Lógicos, a ordem é 'NÃO', 'E' e 'OU'.
